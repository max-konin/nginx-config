# Nginx::Config

Gem implement generator for generate config file for unicorn server instance for nginx

## Installation

Add this line to your application's Gemfile:

    gem 'nginx-config'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nginx-config
